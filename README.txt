== Description ==

This is a custom built plugin for RGPD conformity certificates for french entreprises and it uses INSEE.fr API to retrieve companies data base on SIRET numbers.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `mediateur-search` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
